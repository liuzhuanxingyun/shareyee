## 2. 运行规则

- 环境: Node.js 18+（建议 Node.js 20+）+ npm
- 启动: `npm install` 然后 `npm start`
- 文件说明:
  - `data/portfolio.private.json`: 私有持仓文件，已加入 `.gitignore`
  - `data/portfolio.example.json`: 持仓模板文件
  - `.env`: 本地环境变量文件，用来放端口、接口密钥等配置
  - `.env.example`: 本地环境变量模板文件
