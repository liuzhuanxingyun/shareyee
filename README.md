## 2. 投资看板板块

主要在美股开盘后和收盘后观察持仓表现。

### 看板任务

1. 计算每个资产平均成本价（USD）
2. 联网获取最新价格（近似使用对应股票价格）
3. 计算每个持仓的盈亏金额（USD）和盈亏比例（%）
4. 计算持仓总盈亏金额、总盈亏比例
5. 获取 USD/CNY 汇率并将总盈亏折合 RMB
6. 展示 3 条当下世界影响力较高的新闻
7. 每天早上七点半发送持仓盈亏给对应邮箱
